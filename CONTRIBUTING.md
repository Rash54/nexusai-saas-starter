# Contributing to NexusAI

Thanks for your interest in contributing to NexusAI SaaS Starter.

## What to contribute

- Bug fixes in the Community Edition
- Documentation improvements
- New `.env.example` entries
- Docker/deployment improvements
- Frontend UI polish

## What not to contribute

Pro Edition features (AI engine, integrations, billing) are maintained separately and are not part of this open-source repository.

## How to contribute

1. Fork the repository
2. Create a branch: `git checkout -b fix/your-fix-name`
3. Make your changes
4. Open a pull request with a clear description

## Code style

- Backend: follow PEP 8, use type hints
- Frontend: TypeScript strict mode, no `any`
- Keep components small and focused

## Questions

Open a GitHub issue for bugs or questions about the Community Edition.

For Pro Edition support, use the contact on [Gumroad](https://nexusai.gumroad.com).
